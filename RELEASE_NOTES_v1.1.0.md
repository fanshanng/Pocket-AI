# Pocket AI v1.1.0

`v1.1.0` 是从 `v1.0.0` 到当前正式版的完整更新汇总。本次发布不删除既有功能，不改变 OpenAI 兼容 API 参数语义、导航路径或 Android 原生能力调用方式，重点提升聊天渲染、输入区、会话抽屉、设置页、API 配置、Android 键盘适配、代码块查看和项目维护性。

## 更新摘要

- 消息渲染更稳定：Markdown、LaTeX、KaTeX 和代码块渲染完成系统性修复，减少公式空白、行内公式错位、流式抖动、代码块完成后崩溃和渲染 fallback 不稳定问题。
- 代码块体验升级：普通消息中的代码块按实际宽度自动换行，不再依赖横向滑动；右上角新增全屏查看入口，全屏模式保留语法高亮、纵向浏览和横向滑动。
- 对话操作增强：支持长按消息菜单、区域选择、复制、分享、重新生成、用户消息编辑、回复变体保留和分支切换，让历史对话更容易整理和继续。
- 输入区更顺手：优化附件入口、长文本输入、全屏编辑、发送/停止按钮布局、流式期间编辑、自动滚动暂停/恢复，以及状态栏、小窗和键盘场景下的位置表现。
- 会话抽屉更可靠：改进左侧会话抽屉、搜索、批量删除、列表虚拟化、滑动手势、空白区域触控和主页面联动动画，减少误触和卡顿。
- 设置与外观更完整：整理设置子页面导航、API 配置入口、模型选择页、About 页面、版本检查卡片，并支持浅色、深色和跟随系统外观。
- API 配置更清晰：支持多个 OpenAI 兼容配置，按配置缓存模型列表和推理强度候选，提供显式刷新，避免跨配置数据混淆。
- Android 适配更稳：修复输入框在键盘关闭、窗口尺寸变化、小窗和浮动键盘场景下停留在错误位置的问题，并接入原生 IME inset 桥接。
- 项目结构更轻：拆分 copy、theme、conversation、profile、release、native helper、icon 和 presence 相关逻辑，清理无用依赖和不可达 UI，减少入口文件体积。
- 版本元数据完成更新：应用版本为 `1.1.0`，Android `versionCode` 为 `34`，最新版本检查指向 `HDdssX/Pocket-AI`。

## 主要变化

### 聊天渲染

- 普通 Markdown 继续使用原生渲染，公式内容使用 KaTeX WebView 承载排版。
- 内置离线 KaTeX CSS、字体和运行时资源，降低公式渲染对网络环境的依赖。
- 保留公式分隔符语义，稳定 `\(...\)`、`$$...$$` 和 `\[...\]` 的行内/行间表现。
- 增加公式渲染失败或无响应时的 fallback，避免消息区域出现大片空白。
- 流式期间使用更稳的渲染路径，并节流自动滚动，减少输出过程中的跳动。

### 代码块

- 改为原生 token 高亮，移除不再需要的第三方高亮依赖。
- 修复代码输出完成后崩溃的问题。
- 优化深色表面、长代码行、复制操作和普通消息中的可读性。
- 新增全屏查看模式，适合查看较长代码或保留横向结构的内容。

### 对话与输入

- 新增长按消息操作、区域选择底部面板、复制、分享和重新生成。
- 支持编辑用户消息、保留回复变体，并在分支间切换。
- 重构输入区为附件入口加输入框布局，提升长文本输入和全屏编辑体验。
- 流式回复期间保持输入可编辑，并改善发送、停止、附件和编辑入口的层级。
- 自动滚动支持暂停与恢复，避免阅读历史消息时被新 token 强制拉回底部。

### 会话抽屉与导航

- 会话管理迁移为左侧抽屉，并支持搜索、批量删除和更高效的列表渲染。
- 主页面与抽屉共享滑动动画，减少打开、关闭和窗口变化时的位置不同步。
- 改善抽屉遮罩、列表空白区域和短距离滑动的触控捕获。
- 设置子页面、返回手势和方向转场完成整理，降低页面切换时的突兀感。

### 设置、模型与版本检查

- API 配置移动到设置子页面，模型选择和推理强度候选按配置隔离缓存。
- 模型列表与推理强度候选改为显式刷新，减少页面切换时的错误显示。
- About 页面整理维护者信息、版本展示和更新检查入口。
- 外观设置支持浅色、深色和跟随系统，并适配系统方向旋转。

### Android 与原生能力

- 保留图片分享、多图分享、分屏拖入、附件读取、SecureStore/Keystore 和本地加密存储能力。
- 移除重复键盘避让逻辑，避免键盘关闭或窗口变化后的过期测量继续影响输入区。
- 接入基于 `WindowInsetsCompat` / `WindowInsetsAnimationCompat` 的 Android IME inset 桥接，让输入区跟随真实键盘高度变化。
- 修复浮动键盘、小窗、全高度键盘和运行时窗口尺寸变化下的输入区错位。
